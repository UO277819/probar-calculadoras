Calculadora RPN

Funcionamiento

La calculadora es como si fuese una Pila(una lista de Objetos), es decir, se van introduciendo los elementos con el boton enter, y el orden en el que se operan es de abajo a arriba, por lo que al realizar una operacion se ultilizaran los ultinmos elementos añadidosy al realizarlas se sacaran tantos elementos como sean necesarios para llevarla a cabo. Por ejemplo a la hora de la suma se usaran los ultimos dos elementos de la pila, y el resultado se guardaria en la Pila siendo este el último eleemnto.

Ejemplos:

2+2 Las acciones a realizar son las siguientes: 2 enter 2 enter +

2/(3+1) Las acciones a realizar son las siguientes: 2 enter 3 enter 1 enter + /
